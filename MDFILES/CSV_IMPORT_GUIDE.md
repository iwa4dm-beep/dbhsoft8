## 📋 CSV কলাম বর্ণনা

### **আবশ্যক ফিল্ড (Mandatory) - তারকা (*) চিহ্ন সহ**

| কলাম নাম | ডেটা টাইপ | বিবরণ | উদাহরণ |
|---------|---------|-------|--------|
| **name** | টেক্সট | প্রোডাক্টের সম্পূর্ণ নাম (২+ অক্ষর) | Elegant Evening Abaya Black |
| **brand** | টেক্সট | ব্র্যান্ড নাম (২+ অক্ষর) | DUBAI BORKA HOUSE |
| **categoryid** | আইডি | ক্যাটেগরি আইডি (আপনার সিস্টেম থেকে) | 5f5f5f5f5f5f5f5f5f5f5f5f |
| **fabric** | টেক্সট | ফ্যাব্রিক ম্যাটেরিয়াল | Crepe, Chiffon, Cotton, Silk |
| **color** | টেক্সট | রঙ (২+ অক্ষর) | Black, Navy Blue, White |
| **sizes** | টেক্সট (সেমিকোলন দ্বারা আলাদা) | সাইজারেঞ্জ | L,XL,XXL অথবা XS,S,M,L,XL |
| **costprice** | সংখ্যা | খরচ মূল্য (০ বা বেশি) | 500 |

### **বাধ্যতামূলক কিন্তু ডিফল্ট মান আছে (Optional in CSV)**

| কলাম নাম | ডিফল্ট মান | বিবরণ | উদাহরণ |
|---------|----------|-------|--------|
| **sellingprice** | 0 | বিক্রয় মূল্য (০ বা বেশি, খরচ মূল্যের চেয়ে কম হতে পারে না) | 750 |
| **currentstock** | 0 | বর্তমান স্টক সংখ্যা | 50 |
| **minstocklevel** | 0 | ন্যূনতম স্টক লেভেল | 5 |
| **maxstocklevel** | 100 | সর্বোচ্চ স্টক লেভেল | 100 |

### **ঐচ্ছিক ফিল্ড (Optional)**

| কলাম নাম | ডেটা টাইপ | বিবরণ | উদাহরণ |
|---------|---------|-------|--------|
| embellishments | টেক্সট | সজ্জা/নকশা | Embroidered, Beads, Plain |
| occasion | টেক্সট | উপলক্ষ | Party Wear, Daily Wear, Formal, Wedding, Eid Special |
| barcode | টেক্সট | বারকোড (অনন্য হতে হবে) | BBOX001, UPC123456 |
| productcode | টেক্সট | পণ্য কোড (অনন্য হতে হবে) | PR001, PROD001 |
| madeBy | টেক্সট | নির্মাতা | U.A.E, Bangladesh, India |
| stocklocation | টেক্সট | স্টক লোকেশন | BOX-1, BOX-2, SHELF-A |
| description | টেক্সট | বিবরণ/Notes | Premium evening wear, Summer collection |
| model | টেক্সট | মডেল নম্বর | Abaya-001, Saree-101 |
| style | টেক্সট | স্টাইল প্রকার | Dubai Style, Modern, Casual, Formal |
| isactive | true/false | সক্রিয় অবস্থা | true বা false |

## 📥 ইম্পোর্ট পদ্ধতি

### **ধাপ ১: CSV ফাইল প্রস্তুত করুন**
1. এক্সেল বা Google Sheets এ নতুন ফাইল খুলুন
2. প্রথম সারিতে কলাম শিরোনাম লিখুন (উপরের তালিকা অনুযায়ী)
3. দ্বিতীয় সারি থেকে ডেটা যোগ করুন

### **ধাপ ২: সেভ করুন এবং ফরম্যাট চেক করুন**
- ফোর্ম্যাট: **CSV (কমা-বিভাজিত মান)**
- এনকোডিং: **UTF-8** (বাংলা অক্ষরের জন্য)
- লাইন এন্ডিং: **LF** (Unix/Mac) বা **CRLF** (Windows)

### **ধাপ ৩: ইম্পোর্ট করুন**
1. **Inventory** পৃষ্ঠায় যান
2. **CSV Batch Import** বোতাম ক্লিক করুন
3. আপনার CSV ফাইল নির্বাচন করুন
4. সিস্টেম প্রতিটি পণ্য যাচাই করবে এবং যোগ করবে

## ✨ ভ্যালিডেশন নিয়ম

### **ক্যাটাগরি আইডি খুঁজে পাওয়া**
```
1. Inventory > পণ্য তালিকা-এ যান
2. যেকোনো পণ্য সম্পাদনা করুন
3. "ক্যাটাগরি" ড্রপডাউন খুলুন
4. ডিভেলপার টুলস (F12) খুলুন
5. <option value="XXXX"> এ XXXX হল ক্যাটাগরি আইডি
```

### **স্বয়ংক্রিয় মান নির্ধারণ**
| ফিল্ড | খালি থাকলে | নিয়ম |
|-------|-----------|------|
| sellingprice | ০ সেট হয় | খরচ মূল্যের চেয়ে কম হতে পারে না |
| currentstock | ০ সেট হয় | নেতিবাচক হতে পারে না |
| minstocklevel | ০ সেট হয় | নেতিবাচক হতে পারে না |
| maxstocklevel | ১০০ সেট হয় | ন্যূনতম ১ হতে হবে |
| barcode | স্বয়ংক্রিয় | পণ্য কোড + ৪ সংখ্যা |
| productcode | স্বয়ংক্রিয় | AB + টাইমস্ট্যাম্প (যদি খালি থাকে) |
| madeBy | খালি থাকে | প্রদান করা হয়েছে যদি থাকে |

## 🔍 ত্রুটি পরিচালনা

### **সম্ভাব্য ত্রুটি বার্তা এবং সমাধান**

| ত্রুটি | কারণ | সমাধান |
|------|-------|--------|
| "Product name is required" | নাম খালি | নাম ন্যূনতম ২ অক্ষর লিখুন |
| "Brand required" | ব্র্যান্ড খালি | ব্র্যান্ড ন্যূনতম ২ অক্ষর লিখুন |
| "Category selection is required" | ক্যাটেগরি আইডি খালি/ভুল | সঠিক ক্যাটেগরি আইডি প্রবেশ করুন |
| "Fabric required" | ফ্যাব্রিক খালি | ফ্যাব্রিক নাম লিখুন |
| "Barcode already exists" | বারকোড ইতিমধ্যে ব্যবহৃত | অনন্য বারকোড ব্যবহার করুন |
| "Product code already exists" | পণ্য কোড ইতিমধ্যে ব্যবহৃত | অনন্য পণ্য কোড ব্যবহার করুন |

## 💡 সেরা অনুশীলন

### **✅ করণীয়**
- ✓ সর্বদা হেডার সারি অন্তর্ভুক্ত করুন
- ✓ ফ্যাব্রিক, রঙ, উপলক্ষ এর জন্য সামঞ্জস্যপূর্ণ নামকরণ ব্যবহার করুন
- ✓ সাইজ একাধিক থাকলে কমা দ্বারা আলাদা করুন: `S,M,L,XL`
- ✓ ডেটা যাচাই করুন ইম্পোর্টের আগে
- ✓ ব্যাকআপ রাখুন মূল CSV ফাইলের

### **❌ এড়িয়ে চলুন**
- ✗ হেডার সারি ছাড়া CSV তৈরি করা
- ✗ বিশেষ অক্ষর ব্যবহার করা যা CSV ফরম্যাটে সমস্যা সৃষ্টি করে
- ✗ নেতিবাচক সংখ্যা স্টক বা মূল্যের জন্য
- ✗ খালি/স্পেস সাথে কলাম নাম
