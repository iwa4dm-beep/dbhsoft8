
```
1. Inventory → CSV Batch Import বোতাম
2. আপনার CSV ফাইল নির্বাচন করুন
3. সম্পন্ন! ✨
```

## 📞 দ্রুত সাহায্য

**কোন category ID প্রয়োজন?**  
না! সিস্টেম স্বয়ংক্রিয়ভাবে এটি পরিচালনা করে।

**সীমা কত?**  
কোন সীমা নেই! হাজার হাজার পণ্য ইম্পোর্ট করুন।

**কীভাবে সাইজ একাধিক যোগ করব?**  
কমা দিয়ে আলাদা করুন: `S,M,L,XL`

**কী করব যদি কিছু পণ্য ব্যর্থ হয়?**  
অন্যগুলি যোগ হবে, শুধু ব্যর্থ সারি সংশোধন করুন এবং পুনরায় চেষ্টা করুন।
